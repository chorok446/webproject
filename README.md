## env

```
JWT_SECRET_KEY
MONGODB_URL
AWS_ACCESS_KEY_ID
AWS_SECRET_KEY
AWS_REGION
AWS_BUCKET_NAME
```


## 실행 방법

.env 파일 생성 <br>
위에 적혀있는값 설정 <br>
MongoDB 실행<br>
터미널에 밑에 있는 명령어 타이핑

``` 
npm install 
npm start
```
